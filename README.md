# K-Means Customer Segmentation

This project uses the K-Means Clustering algorithm from Machine Learning to segment customers based on their annual income and spending score.

## Project Description

Customer segmentation is used by businesses to divide customers into different groups based on their purchasing behavior.

In this project, we use the K-Means clustering algorithm to group customers based on:

- Annual Income
- Spending Score

The result is visualized using a scatter plot.

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset

Dataset used: **Mall_Customers.csv**

Features used in this project:

- CustomerID
- Annual Income (k$)
- Spending Score (1-100)
